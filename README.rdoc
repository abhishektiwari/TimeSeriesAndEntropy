== TimeSeriesAndEntropy

Physiological time-series analysis using various entropies such as 
Approximate entropy (ApEn), Sample entropy(SampEn), Multiscale Entropy(MSE).

== Dependencies

* Python 2.5 or higher
* matplotlib
* NumPy
